Core Workflows

This page covers day-to-day use once your actor is loaded.

Navigation And Quick Access Tabs

  • Default pinned tab starts with Core Stats.
  • Maximum pinned tabs: 4.
  • When at max, add actions switch to edit behavior.

To add or remove:

  1. Open the quick tab dialog.
  2. Press a tab from the top list to pin it.
  3. Press a tab from the pinned list to unpin it.

Search And Filters

Search bars are tab-local and filter rows in real time.

  • Type in a tab search bar to hide non-matching rows.
  • Focusing a search bar clears text and resets visibility.

Filter dialogs support 3-state options:

  • Neutral
  • Include
  • Exclude

Workflow:

  1. Open a filter dialog for spells or inventory.
  2. Toggle options until desired state.
  3. Apply to close and run filtering.
  4. Reset to clear selected criteria.

Favorites

Favorites let you pin frequently used items/spells/features.

To toggle a favorite:

  1. Open any row with a favorite icon.
  2. Select the bookmark button.

Expected behavior:

  • Favorite state updates both in source list and Favorites tab.
  • Favorites are actor-specific and synced with the server.

Mode note:

  • In local modes, favorite changes can still be saved locally.

Spellbook: Cast, Slots, Preparation

Cast A Spell

  1. Open a spell row.
  2. Select Cast.
  3. Cast button only appears for prepared spells.
  4. Contextual menu appears with options.
  5. Choosen options are transmitted to the server, rolling attack and/or damage rolls as needed.
  6. Eventual resources consumption is updated by the server then sent to the app.

Contextual Options:

  • Consuming spell slot and/or other resources (when applicable).
  • Using a higher level slot if available.
  • Creating a template on the map.
  • Updating concentration status when applicable.
  • Other options such as (dis)advantage, roll bonus, damage bonus, etc. are a WIP :).

Toggle Spell Slot Directly

  1. In a spell-level group, tap full / empty slot circles.
  2. Confirm Use / Regain prompt.

Prepare Or Unprepare A Spell

  1. Select Prepare on a spell row.
  2. Confirm override prompt if class preparation limit is reached.

Mode Note:

  • If you are connected to a server with the premium module, the updates are sent to the server.
  • Slot/preparation edits are still reflected locally in local modes.

Inventory And Containers

Item Details

  1. Tap an inventory row to expand details.
  2. Read description and tags.
  3. Items that can be "used" (e.g. weapons, consumables, etc.) have an additional "Use" button available.
  4. Similar to casting spells, using items will prompt for resource consumption options.
  5. Items can also be (Un)Equipped.
  6. Use / Equip actions are transmitted to the server and processed accordingly (rolling attacks, damage, displaying item card, etc.).

Open A Container

  1. Expand a container row.
  2. Select Open Container.
  3. Review nested item list under Contents.
  4. Select Back to Info to return.

Note:

  • Containers in containers are unnested for easier navigation.

Conditions

View Condition Description

  1. Tap a condition badge.
  2. Condition dialog opens with name and description.

Remove A Condition

  1. In the condition dialog, select Remove Condition.

Mode Note:

  • If server update is not possible, condition removal will only be applied locally.

Journal

Journal entries are stored per actor and only locally.

Create and edit:

  1. Open Journal tab.
  2. Select New Note.
  3. Enter title and note text.

Additional behavior:

  • Input updates are persisted as you type.
  • Entries can be expanded/collapsed.

Note:

  • Journal entries are local to the actor AND stored in the app cache. They are not sent to Foundry and do not appear in the Foundry Journal tab. We are working on better ways to manage and sync journal content with Foundry in the future.

Health, Temp HP, And Rest

Click on the health bar to open health controls and apply damage, healing, temporary HP, and max HP modifier.

  • Damage subtracts temporary HP first, then current HP.
  • Healing cannot exceed current max HP.
  • Active temporary HP locks input until reset.
  • Max HP modifier are shown in details in the health controls (e.g. "35 / 40 (35+5)") but only final result is displayed in the health bar (e.g. "35 / 40").
  • Max HP modifiers stack on top of each other, you can reset them all at once with the "Reset" button.

Short And Long Rest

  • Open short or long rest dialog from the side panel.
  • Choose options (New Day, Auto-Hit Dice, etc.).
  • Confirm rest action.

Mode Note:

  • The app relies on the server to apply rest mechanics, recover resources, etc. In restricted modes, local data won't change.
